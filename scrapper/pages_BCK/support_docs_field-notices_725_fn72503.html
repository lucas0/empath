Revision History Revision Publish Date Comments 1.0 09-Nov-22 Initial Release Products Affected Affected OS Type Affected Software Product Affected Release Affected Release Number Comments NON-IOS Base Software RI RI.2022.09 Cisco BroadWorks Release Independent Defect Information Defect ID Headline CSCwd39598 NullPointerException thrown when attempting to retrieve device config file on ADP Rel_2022.09_1.344 Problem Description From the DeviceManagement application, a NullPointerException error is thrown when you attempt to retrieve a file. Background From the DeviceManagement application, a NullPointerException error is thrown when you attempt to retrieve a file. This is caused by a change on the Application Delivery Platform (ADP) that is not backward compatible with previous version of the BroadWorksDms application. This issue prevents the Device Management application from doing a route advance if the first repository does not answer.. Problem Symptom This code exception is seen in the DMS logs on the App Server: 500 Internal Server Error. See server logs (Debug severity) for details.
