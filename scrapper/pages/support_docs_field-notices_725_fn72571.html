Revision History Revision Publish Date Comments 1.0 30-May-23 Initial Release Products Affected Affected Product ID Comments NC55-PWR-3KW-AC NC55-PWR-3KW-AC= Part Alternate Defect Information Defect ID Headline CSCvz99098 NCS5500: NC55-PWR-3KW-AC PEM goes to failure on OIR Problem Description The power supply unit (PSU) NC55-PWR-3KW-AC might display error messages, as shown in the Problem Symptom section. The PSU will enter into a failed state with an output disabled alarm during Online Insertion or Removal (OIR) or the installation process. Background This issue was identified during the manufacturing test process and necessary changes have been completed in order to address the problem. This issue does not qualify for a system hardware replacement. There is a software fix available, described in the Workaround/Solution section, to address the problem. Problem Symptom The PSU might show these error messages which help to identify the issue: The admin show environment power command shows that the PSU is in the failed or no power state. sysadmin-vm:0_RP1# show environment power location 0/PM7 DDD MMM DD HH:MM:SS.936 UTC+00:00 0/PM7 3kW-AC 326.1/0.0 /0.0 20.9/0.0 /0.0 10.4 89.1 FAILED or NO PWR The show alarms command displays multiple alarms, which includes Power Module Output Disabled. sysadmin-vm:0_RP1# show alarms DDD MMM DD HH:MM:SS.936 UTC+00:00 ------------------------------------------------------------------------------- Active Alarms ------------------------------------------------------------------------------- Location Severity Group Set time Description ------------------------------------------------------------------------------- 0/FT2 major environ 10/21/21 10:03:25 Fan tray is removed from chassis. 0/PM7 major environ 10/21/21 11:19:11 Power Module Output Disabled. 0/PM7 major environ 10/21/21 11:19:17 Power Module Fault (LOGIC_CTRL_VOLT_OOR). 0/PM7 major environ 10/21/21 11:19:17 Power Module Fan Fault. Workaround/Solution Cisco recommends to install a Software Maintenance Upgrade (SMU) for the respective Cisco IOS® XR release in order to fix the issue. This is also classified as a mandatory SMU for this PSU to be installed now to avoid this issue being seen in future. Note: If you are on Cisco IOS XR Release 7.8.1 or later you do not need to install this SMU as the fix is already built into the release.
