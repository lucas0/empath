Prerequisites Before you start, ensure that you have the following: Active Cisco.com account Universal Device Identifier (UDI) Adding a Device to a Cisco Account in LRP Step 1: Go to Cisco Software Central and log in with your Cisco.com account. Step 2: In the Traditional Licenses section, click Access LRP. Step 3: On the Devices tab, click Add Devices. Note: If the system displays the Welcome To The License Registration Portal alert dialog box, you can select Do not show this message again to prevent it from being displayed again. Step 4: In the Add Device dialog box, from the Product Family drop-down list, select the correct Product Family. Enter the relevant Device Identifier. Click OK. Step result: The system displays the Device Added Successfully notification message. Note: The information requested on the Add Device tab varies depending on a product for which you issue the license key. To obtain the product-specific information that the system requests in the Add Device dialog box, refer to How to Locate the Universal Device Identifier (UDI). Troubleshooting If you experience an issue during the process of adding a device to a Cisco account in LRP, open a case inSupport Case Manager (SCM).
