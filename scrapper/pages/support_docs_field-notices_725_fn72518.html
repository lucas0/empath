Revision History Revision Publish Date Comments 1.0 13-Dec-22 Initial Release Products Affected Affected OS Type Affected Software Product Affected Release Affected Release Number Comments NON-IOS Application Patches 23.0 23.0.2022.10, 23.0.2022.11 Cisco BroadWorks Release 23 NON-IOS Application Patches 24.0 24.0.2022.10, 24.0.2022.11 Cisco BroadWorks Release 24 Defect Information Defect ID Headline CSCwd68410 [BW Eng] BYO PSTN Billing Support for Call-Back and Call-In Problem Description Unable to apply the Application Server (AS) patch ap384015 due to an upgrade script issue. Background When you attempt to apply the AS patch ap384015 with 100 or more Voice XML users, the upgrade script will fail. Problem Symptom This is caused by an internal database read and write sequence error which leads to a SQLException "Function sequence error". Once this issue occurs, the upgrade script fails and the patch is not applied successfully. Workaround/Solution Patches AP.as.23.0.1075.ap384015 and AP.as.24.0.944.ap384015 have been recalled. In order to resolve this issue, customers should download and run a new version of the patch published on Cisco Software Download with a new MD5sum.
