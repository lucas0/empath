Install and Upgrade TechNotes Upgrade FindIT Network Manager or Probe to the Latest Version Save Translations Download Print Updated:December 12, 2018 Document ID:SMB5698 Bias-Free Language Objective New software version releases of the FindIT Network Management have advanced features that would keep both the Probe and the Manager up-to-date with the latest releases and technologies like enhanced security, new features, bug fixes, or performance upgrades. The performance of the FindIT Network Manager and Probe can be enhanced when used with upgraded software version. This process is also necessary if you encounter the following: Frequent network disconnection or intermittent connection on your devices Slow connection This article provides instructions on how to upgrade the software on both components: FindIT Network Manager and FindIT Network Probe Applicable Devices FindIT Network Manager FindIT Network Probe Software Version 1.1 Upgrade FindIT Network Manager or Probe Step 1. Log in to the Administration GUI then click the firmware upgrade button on the upper-right corner of the page. Step 2. To upgrade the Manager to the latest version, click the Manager radio button. Note: The Manager has to be upgraded first before the Probe. In this example, the current version is 1.1.0.2017630 and the latest available software version is 1.1.0.20170711. Note: If you want to upgrade the Probe, click the latest software version to start the upgrade process. Step 3. Click Upgrade. Step 4. (Optional) The system will start to download the latest version. Click Cancel to cancel the upgrade process. Once the download is complete, you will be notified of the status of the upgrade process. The page will automatically close once the upgrade process is complete. Step 5. (Optional) To verify the updated version of the platform, click the button. Step 6. (Optional) To submit upgrade request to associated Probes, click the Upgrade button. You should now have successfully upgraded your FindIT Network Manager or Probe to the latest version.
