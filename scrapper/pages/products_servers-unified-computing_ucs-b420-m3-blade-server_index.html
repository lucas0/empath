The UCS B420 M3 is a balanced, high performance platform that complements the UCS four socket blade server portfolio. It offers industry-leading computing density, I/O bandwidth, and memory footprint, The full-width UCS B420 M3 supports 1.5 TB of memory and takes full advantage UCS virtual interface card (VIC) technology for up to 160 Gbps aggregate I/O bandwidth. The UCS B420 M3 is part of the Cisco Unified Computing System (UCS) that combines blade and rack servers with networking and storage access into a single unified system. Centrally configured through unified, model-based management, Cisco UCS simplifies and speeds deployment of enterprise-class applications running in bare-metal, virtualized, and cloud-computing environments. A Cisco UCS 5108 Blade Server Chassis can house up to four Cisco UCS B420 M3 Blade Servers, supporting up to 6 TB of memory per chassis.
