What's New in Cisco IOS XR Release 7.2.1 Software Fault Profiles The Fault Profiling feature enables you to create a unique fault profile for faults on the system. Each fault profile can contain one or more faults with user-defined severities. See Fault Profiles. Disable auto FPD upgrade before upgrading the software from Release 6.3.x, Release 6.5.x, Release 7.0.1, and Release 7.1.1 to Release 7.2.1.
