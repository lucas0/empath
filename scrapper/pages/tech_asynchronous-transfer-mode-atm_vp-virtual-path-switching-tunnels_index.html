Technology Support VP (Virtual Path) Switching and Tunnels Permanent Virtual Path (PVP): a set of Permanent Virtual Circuits (PVC). A PVC is a virtual link across an ATM network between two specified end points. PVCs are permanently defined by the network manager and are the ATM equivalent of a leased telephone line.
