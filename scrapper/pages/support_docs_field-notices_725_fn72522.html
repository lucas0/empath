Background The FMC MonetDB event database might crash due to a file size error. Once the MonetDB crashes due to the file size error, subsequent restarts of the MonetDB event database might result in the same crash condition and a loss of connection events might occur. Problem Symptom The error messages contained in the "merovingian.log" file on the FMC indicate "BBPcheckbats: !ERROR: xxxx ..7657.tail1 too small (expected 306187, actual 5033)" after the MonetDB crash. /var/log/monetdb/merovingian.log: … 2022-01-10 12:15:15 ERR eventdb[28136]: #client37788:!ERROR: BBPcheckbats: !ERROR: xxxx ..7657.tail1 too small (expected 306187, actual 5033) Workaround/Solution Cisco recommends to upgrade the Firepower software to one of the versions shown in this table.
