Revision History Revision Publish Date Comments 1.0 31-Oct-22 Initial Release Products Affected Affected OS Type Affected Software Product Affected Release Affected Release Number Comments NON-IOS Firepower Management Center Software 7.0 7.0.0, 7.0.0.1, 7.0.1, 7.0.1.1 NON-IOS Firepower Management Center Software 7.1 7.1.0, 7.1.0.1, 7.1.0.2 Defect Information Defect ID Headline CSCvz69699 FMC UI may become inaccessible due to connection leaks in internal database Problem Description The user interface of the Firepower Management Center (FMC) might become inaccessible when integrated with the Cisco Identity Services Engine (ISE). Background There is an FMC database connection leak that might cause usability issues when the FMC downloads the user information from the ISE server. The FMC user interface becomes inaccessible when the connection leak exceeds the internal database resources. Problem Symptom The FMC user interface is not accessible. Workaround/Solution Workaround SSH into the FMC as an admin user and enter the expert shell. FMC> expert Elevate to the root user and restart the Tomcat process. FMC$ sudo su - FMC# pmtool restartbyid Tomcat Solution Cisco recommends to upgrade to one of the fixed Firepower software versions shown in this table.
