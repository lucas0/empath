You would need to transfer licenses between Virtual accounts in the following scenarios: The license was deposited to the incorrect Virtual account by accident. You see a shortage of licenses in one Virtual account and you have abundance of licenses in another Virtual account. Prerequisites Active Cisco.com account Active User or Admin access to a Smart account or/and Active User or Admin access to the original and target Virtual account Transferring Licenses Between Virtual Accounts Step 1: Go to Cisco Software Central and log in with your Cisco.com account. Step 2: In theSmart Software Managersection, clickManage licenses. Step 3: From the menu bar, select Inventory, and then click the Licenses tab. Step 4: To transfer licenses between Virtual accounts, you have the following options: In bulk by selecting the check box for all licenses next to the License column. By individual license by first expanding the plus icon next to the license that you need to transfer, and then selecting the check box for the specific stock keeping unit (SKU). Step 5: From the Available Actions drop-down list, select Transfer. Step 6: From the Select Virtual Account drop-down list, select the Virtual account to which you need to transfer your licenses. Step 7: Select the amount of licenses that you need to transfer in the Transfer column, and then click the Transfer button. Step result: The system displays the License(s) Successfully Transferred message. Troubleshooting If you experience an issue during the process of transferring licenses between Virtual accounts, open a case inSupport Case Manager (SCM).
