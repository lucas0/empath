Products Affected Affected Product Name Description Comments N9K-C9508-FAN Fan Tray for Nexus 9508 chassis, Port-side Intake N9K-C9516-FAN Fan Tray for Nexus 9516 chassis Defect Information Defect ID Headline CSCvw36279 N9500 fan flow direction reversed Problem Description The direction of airflow for some fan trays listed in this notice is incorrect. Background The Nexus 9500 Series supports front-to-back airflow (port side intake). The fans in a small batch of fan trays were installed incorrectly, which causes the airflow to change direction. The show environment fan command output will show front-to-back, however the actual flow might be back-to-front. Problem Symptom The direction of airflow for some fan trays listed in this notice is incorrect. Workaround/Solution Cisco recommends that the affected Nexus 9508/9516 fan trays be replaced. Use the link in the Upgrade Program section to request the replacement parts. Reference Links Nexus 9508 Hardware Installation Guide – Replacing a Fan Tray Nexus 9516 Hardware Installation Guide – Replacing a Fan Tray How to Identify Affected Products This affects only a subset of N9K-C9508-FAN and N9K-C9516-FAN fan trays. Enter the show inventory fans command in order to check the Product ID (PID) and corresponding serial number (SN).
