Licensing Information How to Obtain a License Request for Cisco Unified Communications Manager (CUCM) Save Translations Download Print Updated:May 18, 2021 Document ID:LIC217144 Bias-Free Language Contents Prerequisites Obtaining a License Request for Cisco Unified Communications Manager (CUCM) Prerequisites Before you start, ensure that you have installed licenses on the Cisco Prime License Manager (PLM). Obtaining a License Request for Cisco Unified Communications Manager (CUCM) Step 1: In Cisco Prime License Manager, select Fulfillment from the Licenses drop-down list. Step 2: On the License Fulfillment page, select Generate License Request from the Other Fulfillment Options drop-down list. Step result: The system displays the License Request and Next Steps dialog box. Step 3: In the License Request and Next Steps dialog box, follow the instructions, and then click Close. Step result: You now have the license request on your clipboard as well as the file with the license request saved on your computer. Contributed by Dejan Puhar Was this Document Helpful?
