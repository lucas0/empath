Revision History Revision Publish Date Comments 2.0 01-Jun-22 Updated the Problem Description Section 1.0 13-May-22 Initial Release Products Affected Affected OS Type Affected Software Product Affected Release Affected Release Number Comments NON-IOS Application Patches 24.0 24.0.2022.02, 24.0.2022.03 Cisco Webex BroadWorks Release 24 Defect Information Defect ID Headline CSCwb18112 Post-Upgrade - ACD Login issues when Agent assigned to Multiple Call Centres Problem Description After patches AP382246 and AP382689 are applied, Open Client Interface-Call Control (OCI-C) CallCenterAgentModifyRequest might fail and generate NullPointerException. Background After patches AP382246 and AP382689 are applied, OCI-C CallCenterAgentModifyRequest might fail and generate NullPointerException when an OCI-C CallCenterAgentModifyRequest is sent with no CallCenterAgentSettings attribute. Problem Symptom After the faulty patches AP382246 and AP382689 are applied, agents are unable to change their Automatic Call Distribution (ACD) state and join a CallCenter.
