This certification validates your ability to implement basic network applications using Cisco platforms as a base, and to implement automation workflows across network, security, collaboration, and computing infrastructure. With one exam and one training course to prepare, the Cisco Certified DevNet Associate certification gives you the know-how you need and industry recognition that translates into jobs What you'll learn: Possible job roles Network automation engineer Software developer System integration programmer Prerequisites No formal prerequisites, but one or more years of experience with software development including Python programming is recommended.
