It is designed exclusively for the M4 generation of Cisco UCS C-Series Rack Servers and the C3160 dense-storage server. The service profile determines: Number and type of PCIe interfaces, whether virtual network interface card (vNIC) or virtual host bus adapter (vHBA) Identity (MAC address) and World Wide Name (WWN) Failover policy Bandwidth Quality-of-service (QoS) policies Cisco Data Center Virtual Machine Fabric Extender (VM-FEX) The 1227T card implements a Cisco fabric extender that unifies virtual and physical networking into a single infrastructure. It provides virtual-machine visibility from the physical network and a consistent network operations model for physical and virtual servers. Cisco User-Space NIC (usNIC ) The usNIC improves the performance of software applications using the Message Passing Interface (MPI) instead of sockets or other communication APIs. It does this by bypassing the kernel when sending and receiving networking packets. The applications interact directly with the virtual interface card, which improves the networking performance of your high-performance computing application.
