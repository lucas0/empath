Technology Support Small Computer Systems Interface over IP (iSCSI) iSCSI (Small Computer Systems Interface over IP) works by encapsulating SCSI data traffic and transporting them over TCP/IP networks. This allows the Cisco SN 5420 Storage Router customers to access block-level storage anywhere on an IP network.
