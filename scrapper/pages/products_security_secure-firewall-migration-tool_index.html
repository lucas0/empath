Firewalls Cisco Secure Firewall Migration Tool Simplified migration to Cisco Secure Firewall Cisco Secure Firewall Migration Tool enables you to migrate your firewall configurations to the Cisco Secure Firewall Threat Defense. No matter how complex your current firewall policy is, the migration tool can convert configurations from any Cisco Adaptive Security Appliance (ASA) or Firewall Device Manager (FDM), as well as from third-party firewalls Check Point, Palo Alto Networks, and Fortinet.
