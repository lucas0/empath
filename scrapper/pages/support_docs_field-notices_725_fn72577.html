Background When you attempt to install the latest patch bundle of April 2023 for Release 23.0 or Release 24.0, the refresh command fails. This failure indicates that the patch bundle is invalid. Problem Symptom For security reasons, software available from Cisco is now signed. Software Manager needs to be updated to accept the signed patch bundle.
